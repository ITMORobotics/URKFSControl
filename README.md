# URKFSControl

Install dependencies:
```bash
sudo apt install liburdfdom-dev
sudo apt install liborocos-kdl-dev
sudo apt install libtinyxml2-dev
sudo apt install python3-pykdl
```

Build the kdl_parser:
```bash
cd kdl_parser/kdl_parser/
```

Build the kdl_parser:
```bash
mkdir build
cd build
cmake ..
```
